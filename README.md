# Part 4 – AI Solution Design

## Overview

This project presents an end-to-end AI solution design for the **Healthcare** domain, focused on **Medical Image Triage** using deep learning. It was developed as part of an AI Business Analyst assignment covering problem formulation, data planning, model selection, evaluation, and responsible AI thinking.
